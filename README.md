# Hotel & Lodging Lifeline — Management System

A web-based hotel management system built with Python (Flask) and HTML/CSS/JS.

## Features
- Room Booking
- Room Information
- Room Service / Menu Ordering
- Payment (Credit/Debit Card, UPI, Wallet, Cash)
- Hotel Records

## Setup & Run

```bash
pip install -r requirements.txt
python main.py
```

Then open: http://127.0.0.1:5000
